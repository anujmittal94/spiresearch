Welcome to Inspire Searcher

This is a tool to help locate relevant research papers on INSPIRE-HEP using the references of a paper. Search for a paper through its link on INSPIRE, then you can sort and search its references to find potentially useful papers. Create an account for additional options.

Current Options

  • Sort and search references for a paper.
  • Bookmark papers in a readlist.
  • Bookmark papers for specific projects.

Future Plans

  • Automatic citation generation.
  • Extension to other databases.
